ALGORITHMS
═══════════════════════════════════════════════════════════════════════════════

1. Jensen-Shannon Divergence (Distribution Comparison)
   ┌──────────────────────────────────────────────────┐
   │ JS(P||Q) = 0.5×KL(P||M) + 0.5×KL(Q||M)           │
   │ where M = (P+Q)/2                                │
   │ Range: 0 (same) to 1 (different)                 │
   │ Use: Compare key access frequencies              │
   └──────────────────────────────────────────────────┘

2. Kolmogorov-Smirnov Test (Distribution Comparison)
   ┌──────────────────────────────────────────────────┐
   │ KS = max|CDF_P(x) - CDF_Q(x)|                    │
   │ Range: 0 (same) to 1 (different)                 │
   │ Use: Compare latency distributions               │
   └──────────────────────────────────────────────────┘

3. Jaccard Distance (Set Comparison)
   ┌──────────────────────────────────────────────────┐
   │ Distance = 1 - (|A∩B| / |A∪B|)                   │
   │ Range: 0 (same) to 1 (different)                 │
   │ Use: Compare sequence patterns (bigrams/trigrams)│
   └──────────────────────────────────────────────────┘

4. Drift Score Calculation
   ┌──────────────────────────────────────────────────┐
   │ drift_score = 0.4×freq + 0.3×temporal + 0.3×seq  │
   │ Threshold: 0.3 (30%) triggers recommendation     │
   │ Weights: Frequency most important (40%)          │
   └──────────────────────────────────────────────────┘

WORKFLOW EXAMPLE
═══════════════════════════════════════════════════════════════════════════════

# 1. Initialize collectors
collector = get_simulation_event_collector()
analyzer = PatternAnalyzer(training_patterns)
retrainer = get_auto_retrainer()

# 2. During simulation
collector.start_collection(simulation_id)
for event in simulation:
    collector.add_event(SimulationEvent(...))

# 3. After simulation
sim_events = collector.finish_collection()
sim_patterns = SimulationPatternExtractor().extract_patterns(sim_events)

# 4. Analyze drift
drift_report = analyzer.analyze_drift(sim_patterns)
print(f"Drift: {drift_report.drift_score:.2%}")
print(f"Should retrain: {drift_report.should_retrain}")

# 5. Make decision
decision = retrainer.decide(
    drift_score=drift_report.drift_score,
    simulation_event_count=len(sim_events),
    manual_override=False
)

# 6. If retraining
if decision.should_retrain:
    trainer.retrain_from_simulation(sim_events, num_events=1500)
    retrainer.mark_retraining_completed(
        accuracy_before=0.75,
        accuracy_after=0.77  # +2% improvement
    )
