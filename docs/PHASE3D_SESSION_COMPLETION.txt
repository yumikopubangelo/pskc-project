API WORKFLOW DIAGRAM
═══════════════════════════════════════════════════════════════════════════════

Organic Simulation
       ↓
Generates Events
       ↓
[POST /ml/training/simulation-events]
       ├─ Convert to SimulationEvent
       ├─ Extract patterns
       ├─ Get training patterns
       ├─ Analyze drift
       └─ Return drift_score & analysis
       ↓
[GET /ml/training/drift-status]
       ├─ Get current drift
       ├─ Get cooldown status
       ├─ Get event count
       └─ Return comprehensive status
       ↓
User Decision
       ├─ If should_retrain=true:
       │  └─ [POST /ml/training/retrain-from-simulation]
       │     ├─ Validate events
       │     ├─ Check cooldown
       │     ├─ Make decision
       │     ├─ Start background task
       │     └─ Return retraining_id
       │      ↓
       │     [WebSocket /ml/training/progress/stream]
       │     └─ Monitor progress in real-time
       │      ↓
       │     Accuracy improves ✓
       │      ↓
       │     Cooldown activates (24 hours)
       │
       └─ If should_retrain=false:
          └─ Monitor patterns
             (recheck in 24 hours)


IMPLEMENTATION DETAILS
═══════════════════════════════════════════════════════════════════════════════

Request Processing Flow:

POST /ml/training/simulation-events
  ├─ Validate input (non-empty events)
  ├─ Convert to SimulationEvent objects
  ├─ Extract patterns using SimulationPatternExtractor
  ├─ Get trainer.get_training_patterns()
  ├─ Create PatternAnalyzer with training patterns
  ├─ Call analyzer.analyze_drift(sim_patterns)
  ├─ Log results
  └─ Return SimulationEventsResponse

GET /ml/training/drift-status
  ├─ Get training patterns
  ├─ Get auto-retrainer stats
  ├─ Calculate cooldown status
  ├─ Determine next available retrain time
  └─ Return comprehensive DriftStatusResponse

POST /ml/training/retrain-from-simulation
  ├─ Validate event count (≥100 minimum)
  ├─ Get auto-retrainer instance
  ├─ Call retrainer.decide(drift, events, force)
  ├─ Check decision result
  ├─ Mark retraining started
  ├─ Schedule background task
  ├─ Return RetrainingFromSimulationResponse
  └─ Background task:
     ├─ Run trainer.retrain_from_simulation()
     ├─ Update accuracy metrics
     ├─ Call retrainer.mark_retraining_completed()
     └─ Activate cooldown
